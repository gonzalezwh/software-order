AdminApp (admin/js/admin.js)

This class controls the UI for editing MySQL data via the various /admin/ pages in the software-order form.  Before it works it needs the following set up:

  • var requesttype : The ajax request type for this table
  • var tableindex : The primary index of the table
  • function post_gettabledata(data) : Called after initial ajax.
  • function post_displaytable() : You can use this to add extra data to table cells after they have been filled.
  • function getquickeditoptions(name) : When the user clicks on a cell you can optionally display a dropdown, this gives the AdminApp the array of options (see software.js for an example)
